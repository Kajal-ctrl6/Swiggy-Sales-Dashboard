# 📊 Swiggy Sales Dashboard

## 📝 Brief One-Line Summary

An interactive Excel dashboard that analyzes Swiggy sales performance, customer behavior, and regional trends to generate actionable business insights.

---

# 📌 Overview

This project focuses on analyzing Swiggy's sales data using **Microsoft Excel** to uncover revenue trends, customer preferences, and business performance. The dashboard transforms raw food delivery data into interactive visualizations, enabling users to monitor key metrics and make data-driven decisions.

---

# ❓ Problem Statement

Food delivery businesses generate large volumes of transactional data, making it difficult to identify sales trends, customer preferences, and regional performance.

The objective of this project is to:

- Analyze overall sales performance.
- Understand customer purchasing behavior.
- Compare category-wise revenue contribution.
- Identify regional and seasonal trends.
- Support business decisions through interactive reporting.

---

# 📂 Dataset

The dataset contains food delivery transaction records, including:

- Order ID
- Order Date
- Sales Amount
- Customer Ratings
- Food Category
- City
- State
- Order Quantity
- Delivery Details

The data was cleaned, organized, and transformed before visualization.

# 📈 Key Insights

- Total Sales reached **₹53.01M**.
- Bengaluru contributed the highest sales among all cities.
- Average customer rating remained **4.34**, indicating high customer satisfaction.
- Weekend sales, especially Saturdays, generated the highest revenue.
- Non-Veg food contributed **64%** of total sales.
- Regional analysis highlighted significant differences in state-wise sales performance.
- Q1 and Q2 outperformed Q3 in both sales and order volume.
