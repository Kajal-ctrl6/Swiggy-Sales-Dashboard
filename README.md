# Swiggy-Sales-Dashboard
Excited to share a project where I explored Swiggy’s sales journey—turning raw data into a dynamic dashboard that reveals key trends, customer patterns, and regional insights.

🔍 Project Objective

The primary objective of this project is to transform raw food delivery data into meaningful and actionable insights that help in understanding revenue trends, customer satisfaction levels, and overall business performance. By analyzing data across multiple dimensions such as time, location, and product categories, the project aims to support better strategic decision-making through an interactive Excel dashboard.

📊 Project Overview

This project involves end-to-end data analysis using Microsoft Excel, starting from data cleaning and transformation to building a dynamic and interactive dashboard. The dataset includes key variables such as order details, sales amount, ratings, food categories, and geographical distribution.
The final dashboard provides a comprehensive view of performance metrics like total sales, average ratings, category-wise contribution, and time-based trends. Interactive slicers and charts allow users to explore the data efficiently and uncover insights tailored to specific conditions.

📊 Key Insights

Total Sales reached ₹53.01M, with major contributions from metro cities, especially Bengaluru, indicating strong urban demand

Average Rating of 4.34 reflects a consistently positive customer experience across orders

Weekend sales (particularly Saturdays) show peak performance, highlighting higher customer activity during leisure periods

Non-Veg category contributes 64% of total revenue, significantly outperforming Veg, indicating clear customer preference

State-wise sales variation reveals differences in regional demand, which can help in targeted marketing and expansion strategies

Quarter-wise analysis shows that Q1 and Q2 outperformed Q3 in both sales and order volume, suggesting possible seasonal or operational influences
